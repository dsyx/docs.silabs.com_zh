# Performance <!-- omit from toc -->

本节描述了用于改善性能以及网络性能结果的工具。

- [Manufacturing Test Overview](../Documents/AN718/an718-mfg-test-overview.pdf)：提供了将射频测试和特性集成到标准测试流程中的不同选项的高级描述。适用于那些从早期原型开发阶段转向制造生产环境并需要制造测试援助的客户。
- [Manufacturing Test Guidelines for the EFR32](../Documents/AN700.1/an700-1-mfg-test-guidelines-efr32.pdf)：详细介绍了将射频测试和特性集成到标准测试流程中的不同选项。
- [Performance Results for Multi-PAN RCP for OpenThread and Zigbee](../Documents/AN1385/an1385-multi-pan-rcp-performance-test.pdf)：总结了使用 multi-PAN 进行 CPCd 接口测试的性能测试工作和结果，适用于 OpenThread 和 Zigbee 协议的一些测试场景。
- [Mesh Network Performance Comparison](../Documents/AN1142/an1142-mesh-network-performance-comparison.pdf)：回顾了 Zigbee、Thread 和 Bluetooth Mesh 网络，以评估它们在性能和行为上的差异。
- [Thread Mesh Network Performance](../Documents/AN1141/an1141-thread-mesh-network-performance.pdf) (requires login to the customer portal)：详细描述了测试 Thread Mesh 网络性能的方法；结果旨在提供设计实践和原则以及预期的现场性能结果的指导。
